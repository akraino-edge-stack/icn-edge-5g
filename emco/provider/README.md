**PROVIDER**

This folder containts artifacts that are required for deploying basic and common slice applications that include:
	- emco-monitor
	- external-DNS
	- metallb
	- creation of logical clouds (namespaces)
	- certificate manager
	- Free5gc Apps:	
		* mongodb
		* amf
		* nrf
		* nssf
		* webui
	- sdewan crd controller
	- freegc subscriber controller.
